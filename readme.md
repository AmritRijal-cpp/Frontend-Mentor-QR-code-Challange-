### What I learned

I learned how to centralize a child class to his parent class.
I did it with the help of flexbox and min-height and min-width.

```css
#main {
      display: flex;
      align-items: center;
      justify-content: center;
      min-width:  100vw;
      min-height: 100vh;
}
```
